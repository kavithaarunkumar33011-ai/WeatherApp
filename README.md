# Weather App - Glassmorphism UI 🌤️

Real-time weather app with modern glassmorphism design.

## 🚀 Live Demo
[https://kavithaarunkumar33011-ai.github.io/WeatherApp/](https://kavithaarunkumar33011-ai.github.io/WeatherApp/)

## ✨ Features
- Real-time weather data using OpenWeather API
- Glassmorphism UI with floating animations
- Responsive design for mobile & desktop
- Error handling for invalid cities
- Humidity & Wind speed details

## 🛠️ Tech Stack
- HTML5, CSS3, JavaScript ES6+
- Bootstrap 5, Font Awesome
- OpenWeather API, Fetch API, Async/Await

## 📝 What I Learned
- API Integration & JSON parsing
- DOM manipulation & event handling
- CSS specificity: inline styles vs utility classes
- Modern CSS: backdrop-filter, gradients, keyframe animations
- Git & GitHub workflow
